## 3. 路由（App.tsx）
| 路径 | 页面 | 权限 |
|------|------|------|
| `/login` | Login | 公开 |
| `/` | Dashboard | `PageRoute("dashboard")` |
| `/data` | DataCenter | `PageRoute("data")` |
| `/analysis` | Analysis | `PageRoute("analysis")` |
| `/prediction` | Prediction ★ | `PageRoute("prediction")` |
| `/forecast` | Forecast | `PageRoute("forecast")` |
| `/reports` | ReportCenter ★ | `PageRoute("reports")` |
| `/users` | UserManagement | admin |
| `/settings` | Settings | admin |
| `/monitor` | LlmMonitor | admin |

未授权页面访问显示「无权访问」；导航栏由 `Layout.tsx` 按 `canAccessPage` 过滤。详见 [17-用户管理与权限](./17-用户管理与权限.md)。

## 4. API 客户端（api/client.ts）约定
- 统一 `axios` 实例，`baseURL=/api/v1`，timeout 120s。
- **JWT**：登录后 token 存 localStorage，请求自动附加 `Authorization: Bearer`；401 跳转 `/login`。
- **AuthUser** 含 `allowed_pages`；`AuthContext.canAccessPage(page)` 控制前端路由与导航。
- **用户管理 API**：`listUsers`、`createUser`、`updateUser`、`deleteUser`、`resetUserPassword`、`getUserPageOptions`。
- LLM 参数由 `getLlmProvider()/getLlmModel()/getLlmMode()`（localStorage，默认 volcengine/doubao-seed-2-0-pro-260215/normal）自动附加到 generate/chat/revise 请求；`mode` 取值 `normal`（普通）或 `deep_research`（深度研究），可按模块（agent/prediction/report/analysis）独立配置（`ModeSelect` 组件）。
- 二进制导出用 `responseType:'blob'`。
- **新增后端接口时，先在此加函数，再在页面调用**。

## 5. 两个核心页面交互
### Prediction.tsx
左：生成表单(年/月/补充说明) + 历史记录；右：按大类分组的可编辑表格（重要性下拉 1-5、形势判断 textarea、影响下拉）+ 每行「修改意见 + Agent 修订」横排按钮 + 价格预测输入。操作：保存校准（PUT）、导出 Excel（blob 下载）。

### ReportCenter.tsx
左：报告列表；右：摘要 + 各章节（标题可编辑 + 正文 textarea + "Agent 修订"按钮）+ 内嵌表格单元格编辑 + 审核签发 + 数据源。操作：保存校准、导出 Word、章节级 Agent 校准。

### DataCenter.tsx
Tab：**导入**（上传/样例）、**浏览**（数据集列表）、**质量**（校验结果）、**月报表数据**（按回顾月同步派生/联网获取/编辑 web 类表并保存）。月报生成前须在此预置 6 张系统表。

## 6. 状态管理
组件内 `useState`/`useEffect`，无全局状态库。Agent 视觉状态由 `FloatingAgent` 持有并下传。URL `?id=` 用于深链到具体预测表/报告。

## 7. 扩展点
- 新页面：建 `pages/Xxx.tsx` → `App.tsx` 加 Route → `Layout.tsx` 加导航项。
- 富文本编辑（替代 textarea）：已装 `@tiptap/*`，可在章节编辑中接入（见 14-TODO-FE-1）。
- 大包体优化：`vite build` 提示 >500KB，按需做路由级懒加载（见 14-TODO-FE-2）。
