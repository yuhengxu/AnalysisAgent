# 06 · 数据可视化模块

## 1. 职责

- **后端**：把数据整理成统一"图表协议"（`chart_config`）。
- **前端**：用 ECharts 渲染图表；用 tsparticles/canvas 渲染粒子化 Agent。

## 2. 图表数据协议

后端 `AnalyticsService.chart_config()` 与各 skill 返回统一结构：

```jsonc
{
  "title": "国际原油期货价格走势",
  "xAxis": "日期",
  "yAxis": "价格 (USD/bbl)",
  "source": "CNEEI",
  "series": [
    { "name": "Brent", "data": [["2026-05-01", 99.6], ["2026-05-02", 100.1]] },
    { "name": "WTI",   "data": [["2026-05-01", 91.0]] }
  ]
}
```

- `series[].data` 为 `[x, y]` 二元组数组，前端据此构建 ECharts option。
- 支持的 `chart_type`：`price_trend`、`spread`、`balance`、以及预测情景图（agent 内构造）。

## 3. 前端渲染

| 组件/文件 | 职责 |
|-----------|------|
| `components/ChartPanel.tsx` | 接收 chart_config → 渲染 ECharts（折线/柱状） |
| `pages/Dashboard.tsx` | 价格走势、价差卡片 |
| `pages/Analysis.tsx` | 分析结论 + 图表 + 供需对比 |
| `pages/DataCenter.tsx` | 导入后预览图 |

## 5. 月报表格与图表

| 类型 | 渲染方式 | 数据来源 |
|------|----------|----------|
| 表1-1 价格月度变化 | HTML 表格（ReportCenter） | **snapshot** `report_table_snapshots`（derived，由 `price_series` 同步） |
| 表2-1/2-2/2-3/3-1/3-2 | HTML 表格 | **snapshot**（derived 或 web/手工） |
| 其他月报表格 | HTML 表格，可编辑 | LLM 生成 + 人工校准 |
| 月报 PNG 图表 | `ReportChartImage` + Word 嵌入 | **DB** `AnalyticsService` / `ChartExportService` |

月报图表协议与 Dashboard 相同（见 §2）。PNG 由双引擎渲染：

| 引擎 | 模块 | 切换 |
|------|------|------|
| matplotlib（默认） | `chart_matplotlib.py` | `CHART_RENDERER=matplotlib` |
| PyEcharts + Playwright | `chart_echarts.py` + `chart_playwright.py` | `CHART_RENDERER=echarts` |

统一入口：`chart_render.render_report_chart()`（`chart_theme.py` 兼容 re-export）。仅当 `CHART_ECHARTS_FALLBACK=true` 时，echarts 失败才回退 matplotlib（默认 false）。

PyEcharts 使用独立主题模块 `chart_echarts_style.py`（与 matplotlib 样式分离），内置 `app/static/echarts/echarts.min.js`，通过 ECharts `getDataURL` 导出高清 PNG（默认 1400×780 @ 3× ≈ 4200×2340）。

本地 echarts 模式需：`pip install pyecharts playwright && playwright install chromium`

## 4. 粒子化 Agent 可视化

| 文件 | 职责 |
|------|------|
| `components/AgentAvatar.tsx` | 粒子头像渲染（canvas，additive glow） |
| `components/FloatingAgent.tsx` | 右下角悬浮入口 + 打开对话 |
| `utils/logoParticles.ts` / `logoGeometry.ts` | 从 Logo 图采样粒子 |
| `types/agentVisual.ts` | 视觉状态类型（mood/colors/speed/glow/pulse） |
| `.cursor/skills/image-to-particle-logo/` | 粒子化技巧 SKILL（性能要点） |

视觉状态由后端 `AgentService.VISUAL_PRESETS` 下发（idle/thinking/analyzing/predicting/reporting/success…），对话中可用自然语言（"更科技感""红色警示"）实时调节，详见 08。
