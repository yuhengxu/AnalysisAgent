# 01 · 系统架构总览

## 1. 平台定位

面向能源行业（国际原油）的 **AI 数据分析平台**，核心能力：

1. **数据管理**：导入价格 / 供需 / 影响因素数据并入库。
2. **数据分析与可视化**：价格走势、价差、供需对比、月度统计。
3. **油价预测分析表**（核心交付物一）：六大类影响因素逐项研判 + 价格区间预测 → 导出 Excel。
4. **国际油价月报**（核心交付物二）：完整研究报告自动撰写 → 导出 Word。
5. **拟人化 Agent**：粒子化 Logo 入口，自然语言驱动上述能力。

## 4. 关键设计原则

- **API 层只做编排**：解析请求 → 调用服务 → 返回。无业务逻辑。
- **服务层做业务**：DB 读写、聚合计算、文件产出。
- **Skill 层 = "采集权威数据 + 调用大模型 + 结构化产出"**，是两个核心功能的大脑。
- **大模型可降级**：LLM 不可用时所有 skill 自动走"数据库证据 + 规则兜底"，保证始终有产出。
- **结构化内容存储**：预测表/月报均以 JSON 存 `content_json`，前端编辑、后端导出共用同一结构。

## 6. 模块清单与职责一句话

| 模块 | 代码位置 | 一句话职责 |
|------|----------|-----------|
| 数据库 | `core/database.py` + `models/` | 定义表、管理连接与会话 |
| 数据导入 | `services/data_import.py` + `api/data.py` | 文件 → 解析 → 入库 → 质检 |
| 数据处理分析 | `services/analytics.py` + `api/analytics.py` | 统计/价差/查询/图表配置 |
| 数据预测(模型) | `services/forecast.py` + `api/forecast.py` | 统计法情景预测 + 回测 |
| 预测分析表 | `skills/prediction_skill.py` + `services/prediction.py` | LLM 填表 + 导出 Excel |
| 月报生成 | `skills/report_skill.py` + `services/report.py` | LLM 撰写 + 导出 Word |
| 可视化 | `services/analytics.chart_config` + 前端 `ChartPanel` | 图表数据协议与渲染 |
| Agent | `services/agent.py` + `api/agent.py` | 对话、技能编排、外观调节、章节修订 |
| 用户与权限 | `core/security.py` + `api/auth.py` + `api/users.py` + `models/user.py` | JWT 登录、页面权限、用户管理、月报/预测严格隔离 |
| 大模型 | `core/llm.py` | provider 路由、JSON 模式、降级 |
| 可信数据源 | `skills/sources.py` | 权威机构注册表 |
| 日志 | `core/logging.py` | 统一日志输出 |
