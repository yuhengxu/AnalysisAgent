# AnalysisAgent 功能模块使用文档

## 1. 登录与用户权限

### 1.1 管理员首次登录

默认管理员由后端启动时创建，账号密码来自后端 `.env`：

```env
ADMIN_USERNAME=admin
ADMIN_PASSWORD=admin123
```

登录步骤：

1. 打开前端页面。
2. 进入 `/login`。
3. 输入管理员账号密码。
4. 登录后可看到普通业务页面，以及管理员专属的“用户管理、系统设置、监控日志”。

### 1.2 新增普通用户

入口：

```text
用户管理 → 新增用户
```

操作：

1. 输入用户名。
2. 勾选允许访问的页面。
3. 点击“创建用户”。
4. 将初始密码 `qwer1234` 告知用户。

注意：

- 初始密码会以 bcrypt 哈希写入数据库。
- 普通用户只能看到被勾选的页面。
- 管理员不能删除 admin 账号，也不能删除当前登录的自己。

### 1.3 修改用户权限

入口：

```text
用户管理 → 用户列表
```

操作：

1. 找到目标普通用户。
2. 勾选或取消页面权限。
3. 可选择启用或停用账号。
4. 点击“保存权限”。

权限生效方式：

- 用户刷新页面或重新登录后，前端权限更新。
- 后端 API 权限实时通过数据库校验，不依赖 JWT 中的页面权限。

### 1.4 重置密码

入口：

```text
用户管理 → 用户列表 → 重置密码
```

结果：

```text
密码重置为 qwer1234
```

## 2. 总览 Dashboard

入口：

```text
导航栏 → 总览
```

权限 key：

```text
dashboard
```

主要能力：

1. 查看 Brent、WTI、Dubai、Oman 等核心品种的最新价格。
2. 查看近期区间均价和数据量。
3. 查看价格趋势图和价差图。
4. 快速跳转到数据中心、预测分析表、报告中心等核心功能。

使用前提：

- 已导入价格数据。
- 若无数据，图表与指标为空或仅显示占位。

常见操作：

1. 登录后进入首页。
2. 查看近期价格走势。
3. 根据数据情况进入数据中心补充数据，或进入预测/月报页面生成交付物。

## 3. 数据中心 DataCenter

入口：

```text
导航栏 → 数据中心
```

权限 key：

```text
data
```

数据中心包括四类能力：

1. 数据导入。
2. 数据集浏览。
3. 数据质量检查。
4. 月报表数据预置。

### 3.1 上传文件

支持：

```text
CSV
XLSX
```

可选 category：

```text
price
balance
factor
generic
```

操作：

1. 进入数据中心。
2. 选择上传文件。
3. 可手动选择数据类别，也可以让系统根据文件名自动识别。
4. 点击上传。
5. 后端保存原始文件到 `data/raw/`。
6. 后端解析并入库。
7. 前端展示导入结果和质量检查。

### 3.2 导入 yuebao 样例数据

入口：

```text
数据中心 → 导入 → 导入样例数据
```

作用：

- 批量导入 `yuebao/` 下的样例数据。
- 适合首次体验系统。

### 3.3 数据浏览

入口：

```text
数据中心 → 浏览
```

可以查看：

- 数据集名称。
- 数据类型。
- 数据来源。
- 行数。
- 状态。
- 创建时间。

### 3.4 数据质量检查

入口：

```text
数据中心 → 质量
```

当前主要检查价格数据：

- 是否有价格行入库。
- 是否识别到至少两个品种。

后续可扩展 balance / factor 的空值率、必填列、日期连续性检查。

### 3.5 月报表数据预置

入口：

```text
数据中心 → 月报表数据
```

用途：

月报生成前，需要按回顾月预置 6 张动态系统表：

| 表 | 类型 | 说明 |
|---|---|---|
| 表1-1 价格月度变化 | derived | 由价格数据计算 |
| 表2-1 PMI | web | DeepSearch 获取或手工修正 |
| 表2-2 GDP | web | 手工填写，可选大模型预测 |
| 表2-3 供需差 | derived | 由供需预测数据计算 |
| 表3-1 情景预测 | derived | 来自统计预测模型 |
| 表3-2 机构预测 | web | 机构预测或手工录入 |

推荐流程：

1. 选择回顾月。
2. 点击“同步派生数据”。
3. 点击“深度研究联网获取”或手工填写 web 类表格。
4. 保存修正。
5. 再进入报告中心生成月报。

注意：

- 月报生成时只读取 snapshot。
- LLM 不负责填写这 6 张系统表 rows。
- 如果缺表，报告中心会提示确认是否继续。

## 4. 智能分析 Analysis

入口：

```text
导航栏 → 智能分析
```

权限 key：

```text
analysis
```

主要能力：

1. 按条件查询数据。
2. 运行分析任务。
3. 生成分析结论。
4. 返回图表配置并在前端渲染。
5. 可调用大模型对分析结果进行文字化总结。

使用步骤：

1. 进入“智能分析”。
2. 选择或输入分析任务。
3. 配置数据范围、品种或分析口径。
4. 点击运行。
5. 查看分析结果和图表。
6. 必要时通过 Agent 继续追问。

适用场景：

- 查看价格走势。
- 分析 Brent / WTI 价差。
- 分析供需变化。
- 快速生成图表和文字结论。

## 5. 预测模型 Forecast

入口：

```text
导航栏 → 预测模型
```

权限 key：

```text
forecast
```

该模块是纯统计预测，不等同于“预测分析表”。

主要能力：

1. 对单个品种运行统计预测。
2. 生成 baseline / optimistic / pessimistic 三类情景。
3. 查看历史预测结果。
4. 查看回测指标。

使用步骤：

1. 进入预测模型页面。
2. 选择品种，例如 Brent。
3. 点击运行预测。
4. 查看预测结果。
5. 查看回测结果，包括 MAPE、方向准确率等。

当前算法：

```text
月度均值 → 3 月移动平均 + 近 3 月趋势 → baseline
baseline ± 3 → optimistic / pessimistic
```

适用场景：

- 为月报表3-1提供统计情景数据。
- 快速获得价格趋势参考。
- 为后续更复杂模型提供占位框架。

## 6. 预测分析表 Prediction

入口：

```text
导航栏 → 预测分析表
```

权限 key：

```text
prediction
```

预测分析表是系统核心交付物之一。

### 6.1 生成预测分析表

操作：

1. 进入预测分析表页面。
2. 选择年份和月份。
3. 选择品种，默认 Brent。
4. 输入补充说明，可为空。
5. 点击生成。
6. 系统异步或同步生成预测分析表。

后端流程：

```text
PredictionService.generate
  → PredictionSkill.generate
  → gather_evidence
  → LLM / DeepSearch
  → normalize 28 项因素
  → 写入 predictions(user_id=当前用户)
```

### 6.2 查看与编辑

生成后可编辑：

- 每个因素的重要性 1-5。
- 形势判断及支撑指标。
- 对国际油价影响：促涨 / 持平 / 促跌。
- 当前月价格预测。
- 下月价格预测。

### 6.3 Agent 修订单个因素

操作：

1. 找到某个因素。
2. 输入修改意见。
3. 点击 Agent 修订。
4. 系统调用大模型修订该因素的“形势判断及支撑指标”。

### 6.4 保存人工校准

编辑后点击保存，后端会：

```text
PUT /prediction/{id}
status → reviewed
content_json 更新
```

### 6.5 导出 Excel

操作：

```text
点击导出 Excel
```

导出结果：

```text
data/exports/prediction_{id}_{时间戳}.xlsx
```

Excel 特点：

- 按大类分组。
- 重要性被选中的数字加粗并加下划线。
- 影响方向被选中的选项加粗并加下划线。
- 形势判断自动换行。
- 保留价格预测和说明。

### 6.6 数据隔离

每个预测分析表只属于创建者本人。

用户 A 无法查看、编辑、修订、导出、删除用户 B 的预测分析表。越权访问返回 404。

## 7. 报告中心 ReportCenter

入口：

```text
导航栏 → 报告中心
```

权限 key：

```text
reports
```

报告中心是系统核心交付物之一，用于生成国际油价月报。

### 7.1 生成月报前准备

推荐先完成：

1. 导入价格数据。
2. 导入供需数据。
3. 运行预测模型，生成表3-1数据。
4. 到数据中心“月报表数据”预置 6 张系统表。

### 7.2 生成月报

操作：

1. 进入报告中心。
2. 选择展望月。
3. 系统自动推导回顾月。
4. 点击生成月报。
5. 如果 6 张系统表缺失，前端会提示确认。
6. 等待生成完成。

后端流程：

```text
ReportService.generate_monthly_draft
  → ReportSkill.generate 或 ReportUnrestrictedSkill.generate
  → gather_evidence
  → LLM 撰写章节正文
  → ReportTableDataService.load_for_report
  → guard_report_content
  → 写入 reports(user_id=当前用户)
  → 生成图表 PNG
  → 导出 Word
```

### 7.3 查看与编辑月报

可编辑：

- 摘要。
- 各章节标题。
- 各章节正文。
- 表格单元格。
- 审核签发信息。

注意：

- 6 张系统表会在打开和导出时从 snapshot 重新加载。
- 系统表 rows 不建议直接依赖 LLM 生成。
- 表格人工修正应优先在数据中心“月报表数据”维护。

### 7.4 Agent 章节修订

操作：

1. 找到要修订的章节。
2. 输入专家修改意见。
3. 点击 Agent 修订。
4. 后端基于原文、上下文和修改意见调用大模型。
5. 修订结果回填当前章节。

### 7.5 图表预览

报告中心会通过接口加载月报图表 PNG：

```text
GET /reports/{id}/charts/{chart_id}
```

图表包括：

- 图1-1 Brent / WTI 期货价格。
- 图1-4 Brent 期现货价差。
- 图1-5 Brent-Dubai 价差。
- 图1-6 Brent-ESPO 价差。
- 图1-2、图1-3 待持仓数据源完善。

### 7.6 导出

支持格式：

```text
docx
pdf
tex
```

默认 Word：

```text
GET /reports/{id}/export?format=docx
```

PDF 需要系统安装：

```bash
texlive-xetex
texlive-lang-chinese
pandoc
```

### 7.7 数据隔离

每个月报只属于创建者本人。

用户 A 无法查看、编辑、修订、导出、删除用户 B 的月报。越权访问返回 404。

## 8. Agent 入口

入口：

```text
右下角粒子化 Logo
```

主要能力：

| 指令类型 | 示例 |
|---|---|
| 数据分析 | 分析最近三个月 Brent 和 WTI 走势 |
| 统计预测 | 预测 Brent 后续走势 |
| 预测分析表 | 生成 2026 年 6 月油价预测分析表 |
| 月报生成 | 生成 2026 年第 6 期国际油价月报 |
| 联网查证 | 查询最近 OPEC 会议对油价的影响 |
| 视觉调整 | 把 Agent 调成更科技感 / 红色警示 / 更舒缓 |

返回内容可能包含：

- message 文本回复。
- visual 视觉状态。
- charts 图表。
- report_id。
- prediction_id。

前端会根据 `report_id` 或 `prediction_id` 提供跳转入口。


1. 选择大模型 provider。

```text

deepseek
openai
mock
```

```text
normal
```

- API Key 从后端 `.env` 读取。
- 豆包 deep_research 需要配置 `VOLCENGINE_DEEPSEARCH_BOT_ID`。
- mock 模式不联网，走规则兜底。

## 12. 推荐完整使用路径

```text
3. 进入用户管理
4. 创建普通用户
```

## 13. 注意事项

1. 普通用户没有页面权限时，前端不会显示导航；直接访问 URL 会显示无权访问。
2. 后端也会校验页面权限，不能只依赖前端隐藏。
3. 月报和预测分析表按创建者隔离，管理员默认也不能看其他用户的数据。
4. 异步任务按 `task.user_id` 隔离，别人拿到 task_id 也无法轮询。
5. 当前异步任务状态在进程内存中，生产必须单 worker。
6. 月报生成前应先预置 6 张系统表，否则表格可能为空壳。
7. 大模型不可用时系统会降级，不代表生成失败，但内容质量会下降。
8. 导出 PDF 需要系统安装 LaTeX 工具。
