# 04 · 数据处理与分析模块

## 1. 职责

对入库数据做**统计聚合、价差计算、查询、图表数据生成**，为前端可视化、Agent 分析、预测/月报 skill 提供"证据数据"。

- 服务：`backend/app/services/analytics.py`（`AnalyticsService`）
- 路由：`backend/app/api/analytics.py`

> 该服务是**整个平台的数据读取中枢**：forecast、prediction skill、report skill、agent 都依赖它取数。

## 2. 接口

| 方法 | 路径 | 服务方法 | 说明 |
|------|------|----------|------|
| GET | `/analytics/dashboard` | `dashboard_summary` | 各品种最新价/月均价 + 数据量 |
| GET | `/analytics/prices?symbols=&start_date=&end_date=` | `query_price_series` | 价格序列 |
| GET | `/analytics/spread?symbol_a=&symbol_b=` | `calc_spread` | 两品种价差序列 |
| GET | `/analytics/balance?agency=` | `query_balance_forecast` | 机构供需预测 |
| GET | `/analytics/factors?report_month=` | `query_factor_assessments` | 历史影响因素 |
| GET | `/analytics/charts/{chart_type}` | `chart_config` | 图表数据协议 |
| GET | `/analytics/monthly-stats?symbol=&year=&month=` | `calc_monthly_stats` | 单月统计 |

## 3. 核心方法说明

### `query_price_series(symbols, start_date, end_date)`
按品种 + 日期范围查询，返回 `[{symbol, date, price, unit, source}]`，按日期升序。

### `calc_monthly_stats(symbol, year, month)`

从 `price_series` 表计算**指定自然月**的期货价格统计，供月报表1-1、预测/月报 evidence 使用。

**数据来源（优先级）：**

1. **日频** `source = CNEEI`：取该月 `[month 1日, month 末日]` 内全部交易日，计算算术均价、min、max、count。
2. **月频兜底** `source = CNEEI_MONTHLY`：若无日频，取 `trade_date = 该月最后一日` 的单条月均价。

**衍生指标：**

| 字段 | 口径 |
|------|------|
| `avg` | 当月日频均价（或月频单值） |
| `min` / `max` | 当月日频最低/最高结算价 |
| `mom_pct` | **环比**：`(本月 avg − 上月 avg) / 上月 avg × 100` |
| `yoy_pct` | **同比**：`(本月 avg − 去年同月 avg) / 去年同月 avg × 100` |
| `prev_month_avg` | 上月均价（环比基期） |
| `yoy_base_avg` | 去年同月均价（同比基期） |

**回顾月约定（6 月月报示例）：**

生成「2026 年第 6 期」月报时，请求体传入 `review_year=2026, review_month=5`，表1-1 与图1-1 等均基于 **2026 年 5 月** 的库内数据，而非报告日期所在月（6 月）。

### `build_table_price_change_rows(brent, wti, review_month)`

将 `calc_monthly_stats` 的结果加工为**表1-1**行数据（对齐样例 docx 五行），**禁止由 LLM 填写**：

| 行标签 | Brent / WTI 列 |
|--------|----------------|
| `{review_month}月均价` | `avg`，保留 2 位小数 |
| `环比` | `mom_pct`，格式 `+1.22%` / `-3.50%` |
| `同比` | `yoy_pct`，同上 |
| `最高/最低日均结算价` | `{max}/{min}` |
| `峰谷价差` | `max − min`，保留 2 位小数 |

无数据时对应单元格为 `—`。

### `calc_spread(symbol_a, symbol_b)`
按相同日期对齐做差，返回 `[{date, spread, ...}]`。

### `build_spot_market_evidence(review_year, review_month)`
聚合现货专章 evidence（DTD/Dubai/ESPO 月统计、三类价差月均值及环比、`trends` 方向），供 `review_spot` 正文与图1-4～1-6 对齐。

### `dashboard_summary(start_date?, end_date?)`
按用户选定日期区间（默认近 90 日）汇总 Brent/WTI/Dubai/Oman 的区间均价、最新价与数据点数。

### `chart_config(chart_type, **kwargs)`
生成**前端通用图表协议**（见 06）：
- `price_trend`：多品种价格走势
- `spread` / `price_spread_combo`：期货+现货（左轴）+ 价差（右轴），图1-4～1-6
- `balance`：机构供需对比

## 4. 数据流位置

```
price_series / balance_forecasts / factor_assessments  (DB)
        │  read
        ▼
   AnalyticsService  ──► API(analytics)  ──► 前端图表
        │
        ├──► ReportTableDataService.sync_derived（表1-1 写入 snapshot）
        ├──► ForecastService（统计预测取月度序列）
        ├──► PredictionSkill.gather_evidence（取月度统计/价差/供需）
        ├──► ReportSkill.gather_evidence（同上）
        └──► ChartExportService（月报 PNG，只读 DB，不读 Excel）
```

## 5. 月报专用：回顾月与表1-1 数据链

```
数据中心「月报表数据」→ 同步派生数据 { review_year: 2026, review_month: 5 }
        │
        ▼
AnalyticsService._month_bounds(2026, 5)  →  2026-05-01 ~ 2026-05-31
        │
        ▼
price_series (CNEEI 日频, symbol ∈ {Brent, WTI})
        │
        ▼
calc_monthly_stats("Brent"|"WTI", 2026, 5)
        │
        ▼
build_table_price_change_rows(brent, wti, review_month=5)
        │
        ▼
report_table_snapshots.table_price_change  →  月报生成时 load_for_report
```

- **预置阶段**：`ReportTableDataService.sync_derived()` 在数据中心触发，将表1-1 写入 snapshot。
- **生成/打开/导出时**：`ReportTableDataService.load_for_report()` / `_refresh_report_tables()` 从 snapshot 只读加载，不再实时重算。
