## 一句话理解系统

> 用户上传/导入权威油价数据 → 入库（SQLite）→ 分析服务做统计与图表 →
> 两个核心 **skill**（预测分析表、国际油价月报）调用 **大模型（豆包 / DeepSeek，支持普通与深度研究双模式）** + 平台数据 + **可信数据源**
> 生成结构化结果 → 前端在线查看/编辑/人工校准 → 导出 **Excel / Word**。

## 给后续 AI 协作者的提示

0. **先读 [GUIDE.md](../GUIDE.md) 与 [README.MD](../README.MD)**（项目宪法与工作指引）；Cursor 会自动加载 `.cursor/rules/` 约束。
1. **先读 [15-开发规范与模块边界](./15-开发规范与模块边界.md)**，里面有"我要做 X，应该改哪个文件"的速查表。
2. 每个模块文档底部都有「关键文件」与「扩展点」小节，定位代码用。
3. 任务来源统一看 [14-待完善功能清单](./14-待完善功能清单.md)，每条任务都标注了：涉及文件、验收标准、难度。
4. 改动后务必跑：后端 `python -c "from app.main import app"`，前端 `npm run build`。
